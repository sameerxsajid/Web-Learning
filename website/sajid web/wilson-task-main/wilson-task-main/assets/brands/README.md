# Brand Logos

Place official brand logo images in this directory:

- `reebok-logo.png` - Reebok official logo
- `nike-logo.png` - Nike official logo (Swoosh)
- `adidas-logo.png` - Adidas official logo (3 stripes)
- `puma-logo.png` - Puma official logo

**Recommended specifications:**
- Format: PNG with transparent background
- Size: 200-300px width
- Aspect ratio: Maintain original logo proportions
- Quality: High resolution for crisp display

**Where to get official logos:**
- Official brand websites
- Brand style guides
- Wikimedia Commons (for public domain versions)

**Note:** The website currently uses external logo URLs as placeholders. Replace the image URLs in `index.html` with local paths like `assets/brands/reebok-logo.png` once you have the logo files.
